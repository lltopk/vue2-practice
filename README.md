## create project with cli
```shell
npm install -g @vue/cli
# OR
yarn global add @vue/cli
Create a project:

vue create my-project
# OR
vue ui
```

## node scripts
```
npm install
npm run serve
npm run build
npm run lint
```
Customize configuration: See [Configuration Reference](https://cli.vuejs.org/config/).

## libs
```
npm i element-ui
npm install vuex@3
```
